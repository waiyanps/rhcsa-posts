**Manage Local Users and Groups**

Local user နဲ့ group တွေကိုဘယ်လို create, delete, manage လုပ်မလဲ၊ local password policies တွေကိုဘယ်လိုစီမံခန့်ခွဲမလဲဆိုတာကို အဓိကထားပဲပြိးလေ့လာကြမယ်။ ဒီအပိုင်းမှာဆိုရင် 

- linux system တစ်ခုမှာရှိနေတဲ့ user တွေ group တွေရဲ့ရွယ်ချက်တွေ
- super user / root user account ကိုပြောင်းပြီး linux system တွေကိုစီမံခန့်ခွဲတဲ့အပိုင်းတွေ
